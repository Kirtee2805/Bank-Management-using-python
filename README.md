# 🏦 Bank Management System

A simple Bank Management System built with **Python** and **Streamlit**.

---

## Features

- ✅ Create a new bank account
- 💰 Deposit money
- 💸 Withdraw money
- 📋 View account details
- ✏️ Update account details (name, email, PIN)
- 🗑️ Delete account

---

## Project Structure

```
Bank-Management-using-python/
│
├── bank.py        # Core logic (create, deposit, withdraw, update, delete)
├── app.py         # Streamlit UI
├── data.json      # Auto-generated database file (stores account data)
└── requirements.txt
```

---

## Tech Stack

- **Python 3.x**
- **Streamlit** — for the web UI
- **JSON** — as a lightweight local database

---

## How to Run

**1. Clone the repository**
```bash
git clone https://github.com/Kirtee2805/Bank-Management-using-python.git
cd Bank-Management-using-python
```

**2. Install dependencies**
```bash
pip install streamlit
```

**3. Run the app**
```bash
streamlit run app.py
```
